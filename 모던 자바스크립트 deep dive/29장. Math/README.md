## 29.2 Math 메서드

#### 29.2.1 Math.abs

> Math.abs 메서드는 인수로 전달된 숫자의 절대값을 반환한다.

```javascript
Math.abs(-1);           // 1
Math.abs('-1');         // 1
Math.abs('');           // ?
Math.abs([]);           // ?
Math.abs(null);         // ?
Math.abs(undefined);    // ?
Math.abs({});           // ?
Math.abs('string');     // ?
Math.abs();             // ? 
```

#### 29.9.2 Math.round

> Math.round 숫자의 소수점 이하를 반올림한 정수값을 반환한다.

```javascript
Math.round(1.4); // 1
Math.round(1.5); // 2
Math.round('-1.5'); // ?
Math.round('null'); // ?
```
