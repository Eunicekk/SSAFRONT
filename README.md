1. 매주 수요일 저녁에 오프라인으로 모인다. (시간은 저녁 7시부터 2시간 소요 예정)
2. 각자 챕터 1개씩 맡아서 readme적어서 git에 올리기 + 오프라인 모임 때 설명해주기
3. 시간은 모든 모임원 합의 후 변경 가능
